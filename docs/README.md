# Transportation Document Corpus

The notebook downloads a small official NYC transportation-policy corpus into this folder at runtime.

Default documents:
- TLC Annual Report 2024
- TLC Annual Report 2023
- TLC Annual Report 2022
- FHV Congestion Study Report (2019)
- NYC Mobility Report (2019)
- Taxicab Driver Rules

You may replace or expand these documents as long as the corpus remains relevant to NYC taxi / transportation policy and stays within the assignment requirement of 5–10 PDFs.
