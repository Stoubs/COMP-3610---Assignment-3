COMP 3610 – Assignment 3 NYC Taxi Data Processing & Analysis

Overview: This project processes and analyzes NYC taxi data. The goal is
to clean, transform, and analyze large datasets to extract useful
insights.

Objectives: - Clean and preprocess taxi data - Convert data into
efficient formats (Parquet) - Partition data for better performance -
Perform basic data analysis and visualization

Tools Used: - Python - PySpark - Pandas, NumPy - Matplotlib, Seaborn

Project Files: - assignment3.ipynb → main notebook - requirements.txt →
dependencies - docs/ → supporting reports and PDFs

Data: Large files (CSV, Parquet, outputs) are not included in the repo.
These are ignored using .gitignore to avoid size issues.

How to Run: 1. Install requirements: pip install -r requirements.txt

2.  Run Jupyter Notebook: jupyter notebook

3.  Open: assignment3.ipynb

Notes: - Dataset must be added manually - Outputs are generated when
running the notebook - Docs folder provides background information

Summary: This project shows basic data engineering and analysis skills
using real-world taxi data.
